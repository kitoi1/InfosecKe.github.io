Setting Up Your VM Lab

  • Home
  • About
  • Support

Having your own local pentest lab is crucial for becoming a hands-on ethical hacker. Here's how you can build one using free tools and vulnerable machines like Metasploitable, DVWA, and more.

🧰 What You Need

  • ⚙️ Virtualization Software: VirtualBox or VMware
  • 🐉 Attacker Machine: Kali Linux or Parrot OS
  • 💥 Targets: DVWA, Metasploitable 2, VulnHub machines
  • 📡 Network Setup: Host-only or Internal Network

🧪 Example Lab Setup

  1. Download Kali Linux ISO from kali.org.
  2. Import into VirtualBox and configure 2GB RAM, 2 CPUs, and a host-only network.
  3. Download Metasploitable 2 from SourceForge and add it as a second VM.
  4. Ensure both machines are on the same internal network.
  5. Boot both VMs and run netdiscover from Kali to find Metasploitable’s IP.

🧠 Tip:

Use nmap to scan the target VM and identify open services before exploiting them.
