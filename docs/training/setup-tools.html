Essential Tools Setup Guide

  • Home

  • Support

This page guides you through installing the top cybersecurity tools in your local or cloud hacking environment. All commands are tested on Kali Linux and Ubuntu 24.04.

💻 Basic Setup Commands

sudo apt update && sudo apt upgrade -y
sudo apt install git curl wget unzip python3-pip -y

🛠️ Must-Have Tools

1. Nmap

Network scanner for discovery and vulnerability mapping.

sudo apt install nmap
Official Site →

2. Nikto

Web server vulnerability scanner.

sudo apt install nikto
More Info →

3. Gobuster

Directory and DNS brute-forcing tool.

sudo apt install gobuster
GitHub →

4. Burp Suite (Community)

Top web vulnerability testing tool.

sudo apt install burpsuite
Official Site →

5. Sublist3r

Subdomain enumeration tool.

git clone https://github.com/aboul3la/Sublist3r.git
cd Sublist3r && pip install -r requirements.txt

📦 Install All With One Script

Download and run this custom install script built by Kasau:

git clone https://github.com/kitoi1/kasau-tools-installer.git
cd kasau-tools-installer && chmod +x install.sh
./install.sh
View GitHub Project →

🌐 Optional: Add GUI Launchers

If you're using XFCE or GNOME, create `.desktop` entries for launching tools via app menu.

🛑 Troubleshooting Tips

  • If pip3 gives errors, try: python3 -m pip install
  • If GitHub blocks download, add --depth=1 to the clone command
  • For broken installs: try sudo apt --fix-broken install
