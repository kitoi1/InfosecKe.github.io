Cybersecurity has its own unique language. Whether you're studying for a cert or trying to understand a blog post, knowing the terms is essential.

üß† Common Terms You Must Know

Firewall
A security device or software that monitors and filters incoming and outgoing network traffic based on security rules.
Phishing
A type of social engineering attack often used to steal user data, such as login credentials and credit card numbers.
Malware
Malicious software designed to harm or exploit any programmable device, service, or network.
Zero-Day
A vulnerability in software that's unknown to the vendor and has no fix yet, making it a prime target for attackers.
Encryption
The process of converting data into a coded format to prevent unauthorized access.
Penetration Testing
Simulated cyberattacks on a system to check for exploitable vulnerabilities.
2FA (Two-Factor Authentication)
A security process that requires two different forms of identification to access an account.
Botnet
A group of internet-connected devices infected and controlled by malware, often used to carry out large-scale attacks like DDoS.
